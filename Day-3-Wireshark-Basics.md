# 🌐 Day 3: Wireshark Basics (Network Traffic Analysis)

---

## 🎯 Objective

To analyze network traffic and identify communication details like IP addresses, domains, and protocols using Wireshark.

---

## 🛠️ Tools Used

* Wireshark

---

## 📚 Concepts Learned

* **DNS (Domain Name System)** → Used to resolve domain names into IP addresses
* **TCP (Transmission Control Protocol)** → Shows communication between source and destination
* **TLS (Transport Layer Security)** → Encrypted communication protocol

---

## 🔍 Analysis Performed

### 1. DNS Analysis

* Identified domain names requested by the system
* Example: google.com

---

### 2. TCP Analysis

* Identified source and destination IP addresses
* Observed communication between client and server

---

### 3. TLS Analysis

* Observed encrypted traffic
* Identified server name using SNI (Server Name Indication)

---

## 📊 Key Findings

* Source IP address identified
* Destination IP address identified
* Domains accessed by system detected
* Encrypted communication observed using TLS

---

## 🧠 Understanding

Network analysis is based on:

* IP Address → Who is communicating
* Domain → Which service is accessed
* Protocol → How communication is happening
* Behavior → What activity is occurring
